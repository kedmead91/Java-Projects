# Java-Projects
Java projects

Calculator project:
A calculator that takes 
two numbers and a operator
from the user then returns and
prints out the sum and the answer.

Largest Number Project:
Find the largest number in 
a list using the selectionsort
algorithm. 

Selection Sort Project:
Uses Selection Sort algorithm
to sort numbers in a list from 
largest to smallest. 

Rock Paper Scissors Game:
A backend game of Rock Paper
and Scissors, with the user 
and the computer as players.

Address Project:
Backend Address Book type application.
Asks user for address and email details,
it then returns the details back to the 
user  for them to check, if the details 
are correct then the users details are 
saved and then the application closes.
